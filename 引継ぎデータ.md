# Longtianhong 引継ぎデータ

## プロジェクト概要
麻雀の待ちあてゲーム。筒子のみの一色麻雀。
将来的に1対1のタイマン麻雀に発展させる予定。
待ち判定ロジックは本格麻雀にも流用する前提で設計済み。

## 開発環境
- エンジン：Godot 4系（Standard版）
- プラットフォーム：スマホアプリ（2D完結）
- 言語：GDScript
- 画面サイズ：480×854（縦型）

## 仕様書

### 基本ルール
- 牌：筒子のみ（1〜9）
- 山牌：36枚デッキ（各4枚）から重複なし抽選
- 4枚七対子あり
- 七対子・メンツ手の複合解釈あり（両方の待ち牌を合算して正解登録）
- 残り牌が0枚の理論上の待ちも正解として登録し「なし」との両方を正解扱い

### ステージ構成

**チュートリアル（4枚・7枚）**
- テーマ：雀頭＋メンツ、雀頭＋複数メンツの概念
- 問題：全問固定
- ※チュートリアルは最初から選択可能

**1面**
- 手牌：13枚、簡単なテンパイ、ソート済み、時間制限なし
- 問題：100問プールから10問ランダム選出
- クリア条件：10問全正解

**2面**
- 手牌：13枚、綺麗なテンパイ、ソート済み、時間制限なし
- 問題：100問プールから10問ランダム選出
- クリア条件：10問全正解

**3面**
- 手牌：13枚、複雑な形、時間制限20秒／問
- 問題：リアルタイムランダム生成
- クリア条件：20問全正解（1ミスでゲームオーバー）
- 演出：
  - 3問正解→三日月（背景）
  - 6問正解→半月（背景）
  - 10問正解→フルムーン（背景）＋ソートなし発狂開始

### ゲーム進行
- 正解：1秒ポップアップ→左スライドアウト→右スライドイン（次の問題）
- 不正解／時間切れ：ゲームオーバーポップアップ
- クリア／ゲームオーバー後：再チャレンジ・ホームボタン

### 入力UI
- 1〜9のキーパッド＋「なし」ボタン
- 選択中の牌を表示するパネル（SelectedPanel）
- 選択解除ボタン（BtnClear）
- 提出ボタン（BtnSubmit）
- 排他制御：数字選択中→なし無効、なし選択中→数字無効

### 画面構成（全4画面）
1. タイトル画面（Title.tscn）
2. 会話画面（未実装）※ストーリー演出、スキップボタンあり
3. ステージ選択画面（StageSelect.tscn）
4. 問題画面（Game.tscn）

### 問題画面の常設ボタン（TopBar、3つ並列）
- レイアウト切り替え（横1列↔横2列拡大）※未実装
- 設定ボタン※未実装
- ホームボタン（確認ダイアログあり）※未実装

### 設定ポップアップ（未実装）
- BGM音量調整
- SE音量調整
- 言語切り替え（日本語・英語・中国語・韓国語）

## 実装済み内容

### ✅ MahjongLogic.gd（AutoLoadに登録済み）
- `find_waiting_tiles(hand)`：13枚の手牌から待ち牌リストを返す
- `is_complete_hand(hand)` ：14枚が完成形か判定
- `is_seven_pairs(hand)` ：七対子判定（4枚七対子あり）
- `is_normal_hand(hand)` ：通常手判定（雀頭＋4メンツ）
- `can_form_melds(counts)` ：再帰的メンツ構成確認
- `count_tiles(hand)` ：各牌の枚数をDictionary形式で返す
- `generate_hand(count)` ：指定枚数の手牌をランダム生成
- `create_deck()` ：36枚デッキを生成

### ✅ Title.tscn／Title.gd
- タイトル文字「Longtianhong」表示
- Startボタン→StageSelect.tscnに遷移

### ✅ StageSelect.tscn／StageSelect.gd
- Tutorial・Stage1・Stage2・Stage3ボタン
- 各ボタン押下→Game.tscnに遷移（ステージ情報の引き渡しは未実装）

### ✅ Game.tscn／Game.gd
- 手牌生成・表示（HandAreaにLabelで数字表示）
- キーパッド（Btn1〜Btn9、BtnNone）
- 選択中パネル（SelectedPanel）
- クリアボタン（BtnClear）・提出ボタン（BtnSubmit）
- 排他制御（数字←→なしの相互無効化）
- 正解・不正解判定（完全一致チェック）
- ポップアップ（PopupResult）：正解・不正解メッセージ表示
- もう一度ボタン（BtnRetry）：新問題生成
- ホームボタン（BtnHome）：StageSelect.tscnに戻る
